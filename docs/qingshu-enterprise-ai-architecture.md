# 青数 × QingShuClaw 企业级数据 AI 架构图说明

配套图文件：
`docs/qingshu-enterprise-ai-architecture.excalidraw`

## 1. 青数 + QingShuClaw 介绍

青数正在从“语义建模与问数平台”升级为“数据 + Agent 管理控制台”。
它不仅管理数据源、数据模型和数据集，也开始统一治理 Tool、Skill、Agent、权限、版本与发布。

QingShuClaw 是企业桌面 AI 工作台。
它承接青数发布出来的内置 Agent / Skill / Tool，在一线业务场景中完成调用、协作、结果消费与反馈回流。

## 2. 各自职责边界

青数负责：

- 数据源接入、语义建模、数据集治理
- 元数据、术语、标签、权限和品牌隔离规则治理
- Tool 生成、Skill 编排、Agent 配置与发布
- 内置对象目录、安装包、版本、审计与策略控制

QingShuClaw 负责：

- 登录态承接与免登联动
- 青数内置对象的同步、安装、加载与执行
- OpenClaw 运行时承接与桌面对话体验
- 在真实业务场景中调用 Tool、运行 Skill、消费结果
- 把前线使用反馈带回青数，形成治理闭环

## 3. 青数的流程

完整主链路如下：

1. 数据源接入  
   来自数据库、数仓、业务系统、LBS 与字典系统等多类数据源。

2. 语义建模  
   统一管理主题域、维度、指标、实体关系和权限边界。

3. 数据集与知识层  
   沉淀数据集，同时补齐元数据、术语、标签、口径说明和业务语义。

4. Tool 生成  
   基于数据集创建 Tool，生成 SQL、参数 schema、字段约束与元信息。  
   同时接入 normalize、品牌权限追加、返回条数控制、超时控制、日志审计等通用运行逻辑。
   这些与 Tool 绑定的 SQL 会进一步沉淀为数据域知识资产，为后续 AI 自动生成 SQL、自动规划 Tool、自动组装 Skill 提供天然支撑。

5. Skill 管理  
   通过母版 Prompt、toolRefs、角色说明、元数据与版本信息生成 Skill，并绑定到对应 Agent。

6. Agent 治理与发布  
   把 Tool + Skill 组合成可发布的 Agent，通过 managed catalog 对外提供目录和安装包。

## 4. QingShuClaw 的功能作用及与青数的联动

QingShuClaw 的核心作用，不是重复造一个后台，而是把青数沉淀的垂域能力真正带到业务一线。

典型联动方式：

- 登录后同步青数内置 Agent / Skill / Tool
- 将 managed skill 安装包下载到本地 `SKILLs/` 目录
- 在桌面端通过 OpenClaw 运行时执行这些内置能力
- 在调用 managed tool 时自动携带当前登录 token
- 使用字典 normalize、权限控制和后端治理逻辑保障结果可靠

典型业务场景：

- 灵工供给与商家灵活用工需求分析
- 城市 / 品牌 / 门店相关的 LBS 供需流量分析
- 名称不标准时先做字典匹配，再让用户确认
- 结果继续生成报告、PPT、诊断建议并联动浏览器打开

解决的问题：

- 把“会建模、会管数据”的后台能力，变成“业务人员可直接使用”的桌面 AI 能力
- 减少业务用户直接理解底层模型、口径、SQL 的门槛
- 避免 Tool / Skill 分散维护、权限口径不一致、版本不可控
- 打通“后台治理”与“前台应用”之间最后一公里

## 5. 二者如何协调配合，支撑企业级数据 AI 战略

青数与 QingShuClaw 的协同，不是简单的前后端关系，而是“治理中枢 + 应用终端”的双层协作。

青数负责把垂域数据治理成可被 AI 稳定消费的能力资产：

- 数据模型
- 数据集
- 元数据
- Tool
- Skill
- Agent

QingShuClaw 负责把这些能力资产带入真实工作现场：

- 面向具体任务执行
- 面向业务人机协作
- 面向真实反馈验证

最终形成闭环：

`数据源 -> 语义建模 -> 数据集/元数据/术语/标签 -> Tool -> Skill -> Agent -> QingShuClaw 应用 -> 使用反馈 -> 再回流青数持续治理`

这个闭环的价值在于：

- 垂域数据资产可以持续沉淀为可复用 AI 能力
- 权限、口径、版本和策略保持统一
- AI 不再只是问答入口，而成为企业数据运营与业务决策的执行界面
- 支撑企业从“数据可见”走向“数据驱动的 Agent 化执行”
- 同时持续沉淀各数据域的 SQL 与能力知识库，为未来 AI 自动分析、自动生成 Tool / Skill / Agent 打下基础
